Numerical scheme overview
All solvers implement the same physical model — the Equilibrium Dispersive Model (EDM) — but discretise it differently:
Solver Method Spatial parameter Notes
ChroMo Python / C++ FDM — Finite Difference
(Crank-Nicolson implicit)		 Nx = number of grid points Approximates derivatives; needs ~50–200 points for smooth peaks
CADET FVM — Finite Volume ncol = number of volume cells Integrates equations over cells — conservative by construction; 10 cells can match 30+ FDM points

⚠ Nx and ncol are not directly comparable. A CADET run with ncol = 10 typically achieves accuracy equivalent to or better than ChroMo FDM with Nx = 30. Increasing Nx improves FDM accuracy but also increases computation time quadratically. CADET's FVM converges faster — use 10–30 cells for most runs.
